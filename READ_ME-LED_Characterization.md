# Characterization of each LED 

This method requires the spectrum of each LED at different intervals. In the study, the solar simulator had 21 LEDs, that were measured at intervals of 0.1 intensity (Control parameter = 0.1, 0.2, 0.3, ... 1). An example is given for LED1 and LED 2, but the rest has been omitted for confidentiality and data storage. The original directory was as such:

ls -R
